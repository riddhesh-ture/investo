# Investo - Indian Investment Simulator

Investo is a personal project designed to simulate stock and mutual fund investments specifically for the Indian market. It allows users to track virtual portfolios, analyze performance, and understand investment dynamics.

## Features
- **Mutual Fund (MF) Tracking**: Simulate SIPs and lump sum investments.
- **Stock Market Simulation**: Monitor and manage equity portfolios for the NSE/BSE.
- **Portfolio Analytics**: View virtual gains, losses, and asset allocation across Indian financial instruments.

## Setup
This project is built with React and Vite.
